# **🍝 Mig Ristorante — Gastronomia Italiana de Alto Padrão**
Este projeto é uma landing page premium desenvolvida para o Mig Ristorante, focada em uma experiência de usuário fluida e uma identidade visual sofisticada. O objetivo é unir a hospitalidade italiana tradicional à conveniência digital moderna.

## **💎 Visão Geral do Projeto**
O Mig Ristorante foi projetado para exibir serviços culinários de alto padrão. Além de apresentar o menu, a aplicação guia o usuário por uma jornada refinada em três etapas: Pedido Online, Delivery Gourmet e Experiência Presencial.

O foco técnico deste projeto foi a performance em JavaScript Vanilla, polimento de UI/UX e uma responsividade robusta em todas as escalas de dispositivos.

## **🚀 Principais Funcionalidades**
Depoimentos Dinâmicos: Um sistema de carrossel personalizado desenvolvido em JavaScript que renderiza dinamicamente as avaliações dos clientes a partir de objetos de dados estruturados.

**Animações de Scroll (Scroll-Reveal):** Implementação da API Intersection Observer para disparar efeitos elegantes de fade-in e slide-up conforme o usuário explora a página, garantindo uma sensação premium de "aplicativo".

**Seção de Entrega Contextual:** Cards de serviço interativos que explicam a logística da entrega gourmet e o convite para a experiência refinada no restaurante físico.

**Navegação Moderna:** Sistema de navegação totalmente responsivo, com cabeçalho fixo (sticky header) e menu overlay otimizado para dispositivos móveis.

## **🛠️ Tecnologias Utilizadas**
**HTML5:** Marcação semântica para melhor SEO e acessibilidade.

**CSS3:** Layouts avançados utilizando Flexbox e Grid, incluindo Media Queries complexas para uma responsividade "pixel-perfect".

**Vanilla JavaScript:** Renderização de conteúdo dinâmico e manipulação de DOM sem bibliotecas externas (Zero dependências).

## **📸 Créditos & Recursos**
**Fotografia:** Todas as imagens de pratos e retratos foram obtidas no [Unsplash](https://unsplash.com/).

**Iconografia:** Ícones fornecidos por [Font Awesome] (https://fontawesome.com/).

**Tipografia:** Fontes integradas via [Google Fonts] (https://fonts.google.com/).

## **📱 Desafios de Responsividade**
Um dos pontos altos deste desenvolvimento foi o ajuste fino da responsividade para dispositivos de até 480px. As principais soluções incluíram:

Resolução de sobreposição do cabeçalho fixo usando espaçamento dinâmico.

Reestruturação de layouts flex-grid de 4 colunas para visualizações de 2 e 1 coluna, mantendo a hierarquia visual.

Otimização de áreas de toque para usuários de dispositivos móveis.<hr>

# **🍝 Mig Ristorante — High-End Italian Gastronomy**
This project is a premium landing page developed for Mig Ristorante, focusing on a seamless user experience and a sophisticated visual identity. It bridges the gap between traditional Italian hospitality and modern digital convenience.

## **💎 Project Overview**
Mig Ristorante is designed to showcase high-end culinary services. Beyond presenting the menu, the application guides the user through a refined three-step journey: Ordering, Delivery, and Dine-in Experience.

The technical focus of this project was on Vanilla JavaScript performance, UI/UX polish, and robust responsiveness across all device scales.

## **🚀 Key Features**
**Dynamic Testimonials:** A custom carousel system powered by JavaScript that dynamically renders customer reviews from structured data objects.

**Scroll-Reveal Animations:** Implemented the Intersection Observer API to trigger elegant fade-in and slide-up effects as the user explores the page, ensuring a premium "app-like" feel.

**Contextual Delivery Section:** Interactive service cards that explain the logistics of gourmet delivery and the invitation for a refined in-person experience.

**Modern Navigation:** A fully responsive navigation system, featuring a sticky header and a mobile-friendly overlay menu.

## **🛠️ Tech Stack**
**HTML5:** Semantic markup for better SEO and accessibility.

**CSS3:** Advanced layouts using Flexbox and Grid, including complex Media Queries for "pixel-perfect" responsiveness.

**Vanilla JavaScript:** Dynamic content rendering and DOM manipulation without external libraries (Zero-dependencies).

## **📱 Responsive Challenges**
One of the highlights of this development was fine-tuning the responsiveness for devices as small as 480px. Key solutions included:

Resolving fixed-header overlaps using dynamic spacing.

Restructuring flex-grid layouts from 4-column to 2-column and 1-column views while maintaining visual hierarchy.

Optimizing touch targets for mobile users.
